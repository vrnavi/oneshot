                    About Dishwasher #

                    Dishwasher is the primary moderation bot for the OneShot Discord, based off of robocop-ng, with heavy adjustments. You can view its source (and contribute if you wish) here.

                    Some commands can be used by users, others by Staff, and others by the server owner.
